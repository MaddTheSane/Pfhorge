Liquid Definitions


Placing Liquids

Selecting "View Liquids" opens a drawer with any existing light values and switches the map to a color-coded view, matching the polygons to any placed liquids. The presentation of the parameters differs from Forge in some respects:

Type parameter is applied in the same way. Currently you are limited to the liquid type associated with the map environment: no mixed liquids yet.

Copy ("Based On") is unavailable.

Tide parameters are applied in the same way; preprogramme light parameters (Lights) and select from drop-down menu.

Transfer Mode is new and offers the means to apply wobble, slide, etc to the liquid surface.

High and Low Tide parameters are entered as points rather than WU.

Flow Strength is measured in a scale of 0 to 1024 rather than 0 to 1.

Angle of flow is entered numerically; Marathon/Aleph One programming means due West is 0° and the compass has 512°
(West = 0°, North = 128°, East = 256°, South = 384°).

Each liquid can be given a specific name.

Origin X and Y, Height, Min Light and Texture are parameters available within Alph One but are unused as yet.

Limitation:

Tide effects can't be seen in Visual Mode.
