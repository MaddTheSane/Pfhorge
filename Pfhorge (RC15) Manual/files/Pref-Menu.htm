Preferences & Menus Guide



General Prefs

Colours: selects colors to be applied to the different components of the map. Also has facility to turn off the coloring for specialized polygons. The colors can be changed by double-clicking on each rectangle and selecting a new color on the system palette that appears.


Grid: sets properties of the grid the map is built on. The colors are again editable by double-clicking and the grid size varies from 8 to 1/8 world units.


General: options to assist in drawing maps and adding objects. When a line is drawn the endpoints will "snap" to the nearest other point or grid cross-section; the sensitivity of this feature can be altered or disabled, as can the guiding algorithms. New objects can also be set to snap to a grid cross-section while invisible layers (see below) can be set to appear as points rather than empty polygons.


Object Prefs: sets colors for specific object types and determines whether they are visible on the map. The object borders can set to be highlighted for easier observation on the map. See "Objects".


Visual Mode

Environment: Select Shapes enables you to specify the Shapes file you want to use when designing a map. Render Mode: options for how the textures appear in Visual Mode, ranging from full textures through wireframe. Starting Position: choice of player positions to start from. Plain colors: set of colors for use in untextured Render Mode. Other Colors: similar to above.


Visibility: options on what will be shown in Visual Mode and how it will be shown.


Keys: keyboard and mouse settings for manouvering around the map in Visual Mode.


Default Layer Settings: Pfhorge is capable of breaking an imported map into vertically separate layers pre-defined by the user. Each layer can be viewed and worked on separately in map-editting. In this preference panel individual layers can be defined and named, the floor & ceiling values entered with sliders or directly in the boxes. Overlap between the layers is OK as long as the Ceiling Min is greater than the Ceiling Max of the layer below and the the Floor Max less then the Floor Min of the layer above.



Menus

File Menu
New Pfhorge Level: opens a new Pfhorge map.
New Pfhorge Scenario: opens a file in which a list of maps can be assembled to create a scenario. Encompasses all the map files within a selected folder.
Open...: opens a saved map file.
Open Recent: submenu containing a list of the 10 most recently opened maps.
Import Marathon Map: imports maps from Aleph One, Marathon Infinity and Marathon 2 into a Pfhorge map.
Import Pathways Map: imports maps from Pathways into Darkness into a Pfhorge map.
Test Checksum...: tests saved maps for corruption.
Close/Save/Save As...: usual functions.


Edit Menu
Usual Functions


View Menu
View Mode Normal: switches view to the map-editting window.
Visual Mode: opens a new window and creates a 3D visualization of the map. These menu items open a drawer at the side of the map with names/numbers & colors assigned to specific polygon properties and color the map-editting view to match this. New properties are created and assigned to polygons in this way. Submenus allow the selection of floor or ceiling.

View Layers: opens a drawer to view and assign layers to existing polygons. New layers can't be created this way.
View Liquids: changes coloring in map-editting to reflect liquids dispersion on the map.
View Heights: changes coloring in map-editting to reflect floor or ceiling height.
View Lights: changes coloring in map-editting to reflect floor or ceiling lighting or liquid tide.
View Sounds: shows any sounds associated with specific polygons.


Level menu
Toggle Map Manager Drawer: opens drawer on map-editting window to enable quick alteration of some editting features. (See separate entry on Map Manager Drawer.)

Zoom In: zooms in step-by-step in the map.
Zoom Out: zooms in step-by-step in the map.
Zoom To 100%; returns to default zoom view.

Get Info on Selected: produces information on selected line, polygon, object, etc.
Edit Polygon Name: polygons can be given specific names in addition to their index numbers.
Calculate Sides On Selected Lines: trouble-shooting tool.

Go To Via Index: highlight specific lines or polygons according to their index numbers. Mainly used during error tracking.

Marathon Settings Sheet: sets game properties of individual maps.

Redraw Level:

Import Marathon Map...: imports maps from Aleph One, Marathon Infinity & Marathon 2.
Export to Aleph One: exports map to an Aleph One file.


Tools Menu
Tools Palette: opens a palette containing the map-editting tools.

The Inspectors are windows that enable editting of the selected items:
Basic Inspector: holds the edittable variables of the selected line, polygon or object.
Control Panel Inspector: defines the type of control panel and what is controls.
Light Inspector: light intensity of the wall textures.
Texture inspector: editting of the wall textures.

The Editors are more general in their functions:
Terminal Editor: enables the construction of messages attached to the terminal screens.
Configuration Editor: global editor for tags, lights, sounds, platforms, named polygons and layers.
Item/Object Placement Editor: replenishment and placement of aliens and weapons.

Show Colors Palette: Apple's color palettes for selecting colors.


Scripts Menu
Enables the loading and running of scripts to complete repetitive tasks.
