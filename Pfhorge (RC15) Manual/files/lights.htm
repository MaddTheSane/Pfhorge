Light Definitions


Adding Lights to floor and ceilings

Selecting "View Lights" (Floor or Ceiling) opens a drawer with the 20 pre-coded light values and switches the map to a color-coded view, matching the polygons to the respective light values. Double-clicking on a value or creating a new one opens a parameter box similar to Forge's.

A few things are different: Period and Intensity can be set with sliders as well as typing into individual boxes and a Light can be given a specific name.

At present however "Copy From" (Forge's "Based On") is inactive. The pull-down menu that included the strobe effect upon light activation has gone but strobing still exists, the Forge menu choice simply placing preset values in the parameter boxes.

As with Forge, the tidal behaviour of liquids is programmed through lights as if the rise and fall were a change in intensity. See Liquids for further details.


Adding Lights to walls

The light effects of walls are added in Normal Mode through the Light Inspector using parameters created in the Lights:Floor or Lights:Ceiling modes. Since in Pfhorge lights can be given names you can associate a light effect with a particular feature when applying it.
