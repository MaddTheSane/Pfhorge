Configuration Editor


This editor keep track of all tags, lights, sounds, liquids, platforms, named polygons & layers within each map. A drop-down menu allows for the selection of each resource type and a command menu enables some editting to be undertaken. It should be noted that, aside from changing names, most of the editting is done automatically elsewhere and this function exists as a back-up. It serves primarily as a comprehensive reference tool for how the various components within a map are currently interlinked.


By double-clicking on a name it can be editted, providing verbal links to key features in a map. Platforms, lights & liquids can already be given names while they are being defined through their respective "Rename" boxes; this provides an alternative route and extends the feature to tags and sounds.
